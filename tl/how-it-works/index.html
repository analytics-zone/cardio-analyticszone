Paano Gumagana ang CardioAnalytics

Walang putol na pag-sync sa Apple HealthKit para sa real-time na kardyobaskular na pagsubaybay, background updates at write-back capabilities

HealthKit Integration Overview

Ang CardioAnalytics ay gumagamit ng Apple HealthKit upang magbasa at magsulat ng kardyobaskular at mobilidad na datos. Ang lahat ng datos ay nananatiling lokal sa inyong iPhone - walang cloud storage, walang external servers, walang kinakailangang account.

Mga First-Party Data Types

Gumagamit ng opisyal na Apple HealthKit identifiers para sa lahat ng 11 metrics. Gumana nang perpekto sa Apple Watch, fitness trackers, manual inputs at connected medical devices.

Real-Time Syncing

Ang anchored queries at background delivery ay nagsisiguro ng sariwang datos nang hindi inuubos ang baterya. Ang Delta syncing ay nag-pull lamang ng mga bagong sample mula sa huling sync.

Write-Back Support

Ang mga user inputs (timbang, presyon ng dugo, ECG notes) ay maaaring isulat pabalik sa HealthKit upang mapanatili ang consistency sa Apple Health ecosystem.

Paano Ito Gumagana: Hakbang-Hakbang

1

Pahintulutan ang HealthKit

Sa unang paglunsad, hinihiling ng CardioAnalytics ang pahintulot upang magbasa ng kardyobaskular at mobilidad na data types mula sa Apple Health. Maaari kayong pumili ng eksakto kung aling metrics ang ibibahagi.

  • Tibok ng Puso (resting, walking, current)
  • Presyon ng Dugo (systolic/diastolic)
  • Pagkakaiba-iba ng Tibok ng Puso (SDNN, RMSSD)
  • Saturasyon ng Oksihena (SpO‚ÇÇ)
  • Timbang at BMI
  • ECG Recordings at Atrial Fibrillation Detections
  • VO‚ÇÇ Max
  • Mga Mobility Metrics (walking speed, asymmetry, stair speed)

üîí Detalyadong Kontrol: Kayo ang pumipili ng mga data types. Maaari kayong mag-authorize ng lahat o pumili ng specific metrics.

2

Awtomatikong Background Sync

Kapag authorized na, ang CardioAnalytics ay nag-set up ng background observers para sa bawat data type. Kapag bagong samples ang naidagdag sa HealthKit (mula sa Apple Watch, manual inputs o iba pang apps), ang CardioAnalytics ay awtomatikong nag-sync.

  • HKAnchoredObjectQuery - Nag-pull ng bagong samples mula sa huling sync anchor
  • Background Delivery - Pinapagana ang app na mag-update kahit closed
  • Delta Syncing - Nag-pull lamang ng mga bagong datos, hindi nag-refetch ng lahat
  • Battery-Efficient - Walang patuloy na polling; updates ay push-based

üîã Optimized para sa Baterya: Ang background sync ay trigger lamang kapag may mga bagong samples, hindi nag-poll nang patuloy.

3

Visualisasyon ng Datos at mga Alerto

Ang synced data ay agad na makikita sa personalized dashboard. Ang mga uso ay kinakalkula sa loob ng app gamit ang on-device processing - walang cloud uploads.

  • Chart trends para sa bawat metric (lingguhan, buwanan, taunang views)
  • Ikumpara ang mga halaga laban sa guideline ranges (AHA, Mayo Clinic, Cleveland Clinic)
  • Tumanggap ng mga alerto para sa abnormal values (customizable thresholds)
  • Mag-log ng mga sintomas at gamot na nakaugnay sa metric timestamps
4

Mag-export ng mga Ulat

Bumuo ng komprehensibong PDF o CSV na mga ulat upang ibahagi sa inyong doktor. Ang mga ulat ay naglalaman ng lahat ng metrics, medications, symptoms at care goals.

  • Propesyonal na naka-format na PDF na may charts at tables
  • CSV exports para sa data analysis sa spreadsheets
  • Magbahagi sa pamamagitan ng email, Messages o AirDrop
  • Walang data ang nag-leave sa inyong device hanggang sa kayo ay mag-export

Privacy at Seguridad

Ang inyong kardyobaskular na datos ay sensitibo. Narito kung paano pinoprotektahan ng CardioAnalytics ang inyong privacy:

  • 100% On-Device Processing - Lahat ng datos ay nananatili sa inyong iPhone. Walang cloud uploads.
  • Walang Accounts - Walang sign-ups, walang logins, walang user profiles sa remote servers.
  • Walang Tracking - Hindi kami nangongolekta ng analytics, usage data o personal information.
  • Detalyadong HealthKit Authorization - Kayo ang pumipili ng eksakto kung aling data types ang ibibahagi.
  • Kayo ang Nagmamay-ari ng Inyong Datos - Ang mga export ay nangyayari lamang kapag pumili kayo. Walang automated sharing.

üîí Privacy by Design: Ang CardioAnalytics ay dinisenyo mula sa simula upang protektahan ang inyong medical data. Hindi kami maaaring makita ang inyong datos - ito ay umiiral lamang sa inyong device.
