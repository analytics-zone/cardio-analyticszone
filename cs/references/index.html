Zřeknutí se odpovědnosti

CardioAnalytics neposkytuje lékařskou diagnózu nebo léčbu. Všechny metriky jsou pouze pro informační účely. Vždy se poraďte s kvalifikovaným lékařem o lékařské radě, diagnóze nebo rozhodnutích o léčbě.

Reference jsou poskytnuty pro vzdělávací účely k vysvětlení vědeckého základu kardiovaskulárních metrik. Zahrnutí reference neznamená souhlas citovaných autorů nebo institucí.
