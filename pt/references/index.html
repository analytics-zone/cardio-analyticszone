Aviso

O CardioAnalytics não fornece diagnóstico médico ou tratamento. Todas as métricas são apenas para fins informativos. Consulte sempre um clínico qualificado para aconselhamento médico, diagnóstico ou decisões de tratamento.

As referências são fornecidas para fins educacionais para explicar a base científica das métricas cardiovasculares. A inclusão de uma referência não implica endosso pelos autores ou instituições citadas.
