血压监测

使用 AHA 分类标准追踪收缩压和舒张压

什么是血压？

血压是血液推压动脉壁的力量，以毫米汞柱（mmHg）为单位测量。它有两个组成部分：

  • 收缩压 - 上面的数字，测量心脏搏动（收缩）时的压力
  • 舒张压 - 下面的数字，测量心脏在两次搏动之间休息时的压力

示例：120/80 mmHg 的读数表示收缩压 120，舒张压 80。

血压为何重要

持续性高血压（高血压症）是主要的心血管风险因素：

  • 增加中风和心脏病发作（MI）的风险
  • 可能损害血管、心脏、肾脏和其他器官
  • 通常没有症状（"沉默的杀手"）
  • 可通过生活方式改变和药物治疗

血压分类（AHA）

美国心脏协会定义了这些分类（AHA 指南）：

正常

<120 / <80 mmHg

保持健康习惯以将血压维持在此范围内。

偏高

120-129 / <80 mmHg

有发展为高血压的风险。建议改变生活方式。

1 期高血压

130-139 / 80-89 mmHg

请咨询临床医生。改变生活方式，可能需要药物治疗。

2 期高血压

≥140 / ≥90 mmHg

需要医疗治疗。药物和生活方式改变相结合。

高血压危象

>180 / >120 mmHg

紧急情况：立即寻求医疗救助。

CardioAnalytics 如何使用血压数据

  • 存储配对的收缩压/舒张压读数 - 带时间戳的完整血压测量
  • 叠加 AHA 分类区间 - 可视化您的读数属于哪个类别
  • 趋势分析 - 追踪血压在天、周和月中的变化
  • 药物相关性 - 查看降压药如何影响您的血压
  • 偏高读数警报 - 基于 AHA 指南或您的临床医生目标的个性化阈值
  • 回写到 HealthKit - 手动血压输入同步到 Apple Health 以保持一致性

📊 追踪多次读数：血压全天都有变化。每天在同一时间测量以实现一致的追踪。

HealthKit 数据类型

CardioAnalytics 使用这些标识符从 Apple HealthKit 读取血压数据：

  • bloodPressureSystolic - 收缩压（mmHg）（Apple 文档）
  • bloodPressureDiastolic - 舒张压（mmHg）（Apple 文档）

收缩压和舒张压读数使用 HealthKit 的关联 API 进行关联，以确保配对测量。

准确测量血压的提示

  • 使用经过验证的设备 - FDA 批准或临床验证的血压计
  • 每天在同一时间测量 - 早晨和晚上是理想时间
  • 测量前休息 - 测量前安静坐 5 分钟
  • 正确定位 - 手臂与心脏齐平，双脚平放在地板上，背部有支撑
  • 避免咖啡因、运动、吸烟 - 测量前 30 分钟
  • 进行多次读数 - 平均间隔 1 分钟的 2-3 次读数
  • 使用正确的袖带尺寸 - 袖带应覆盖上臂的 80%
