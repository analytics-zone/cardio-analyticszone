Exempció de Responsabilitat

Cardio Analytics no proporciona diagnòstic o tractament mèdic. Totes les mètriques són només amb finalitats informatives. Sempre consulteu un metge qualificat per a consells mèdics, diagnòstic o decisions de tractament.

Les referències es proporcionen amb finalitats educatives per explicar la base científica de les mètriques cardiovasculars. La inclusió d'una referència no implica el suport dels autors o institucions citades.
