血圧追跡

AHA分類による収縮期血圧と拡張期血圧を追跡

血圧とは？

血圧は、動脈壁を押す血液の力で、水銀柱ミリメートル（mmHg）で測定されます。2つの成分があります：

  • 収縮期血圧 - 上の数値、心臓が拍動する（収縮する）ときの圧力を測定
  • 拡張期血圧 - 下の数値、心拍間に心臓が休んでいるときの圧力を測定

例：120/80 mmHgの測定値は、収縮期120および拡張期80を意味します。

血圧が重要な理由

持続的な高血圧（高血圧症）は主要な心血管リスク因子です：

  • 脳卒中や心臓発作のリスクを増加
  • 血管、心臓、腎臓、その他の臓器を損傷する可能性
  • 症状がないことが多い（「サイレントキラー」）
  • ライフスタイルの変更と薬剤で管理可能

血圧分類（AHA）

American Heart Associationはこれらの分類を定義しています（AHAガイドライン）：

正常

<120 / <80 mmHg

血圧をこの範囲に保つために健康的な習慣を維持してください。

血圧上昇

120-129 / <80 mmHg

高血圧症発症のリスク。ライフスタイルの変更が推奨されます。

ステージ1高血圧

130-139 / 80-89 mmHg

医師がライフスタイルの変更と場合によっては薬剤を推奨する可能性があります。

ステージ2高血圧

≥140 / ≥90 mmHg

通常、薬剤とライフスタイルの変更の両方が必要です。医師にご相談ください。

高血圧クリーゼ

>180 / >120 mmHg

緊急医療対応が必要。すぐに救急サービスに連絡してください。

CardioAnalyticsでの血圧データの使用方法

  • AHA分類付きトレンドグラフ - 時間経過による血圧を可視化
  • 高血圧ステージのアラート - 血圧上昇、ステージ1、ステージ2の通知
  • 服薬との相関 - 降圧薬が血圧にどのように影響するかを追跡
  • カスタマイズ可能なしきい値 - 医師の推奨に応じてアラートを調整
  • エクスポート可能なレポート - 医療専門家のための完全な血圧履歴
