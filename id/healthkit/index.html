Integrasi HealthKit

Sinkronisasi mulus dengan Apple Health untuk pemantauan kardiovaskular otomatis

Apa Itu Apple HealthKit?

HealthKit adalah framework Apple yang memungkinkan aplikasi kesehatan dan kebugaran untuk berbagi data dengan Apple Health. Ini membuat Apple Health sebagai repositori terpusat untuk semua data kesehatan Anda dari berbagai sumber—Apple Watch, iPhone, perangkat medis, dan aplikasi pihak ketiga.

Jenis Data yang Didukung Cardio Analytics

Cardio Analytics membaca jenis data kardiovaskular dan mobilitas berikut dari HealthKit:

Metrik Kardiovaskular

  • heartRate - Detak jantung saat ini (denyut/menit)
  • restingHeartRate - Detak jantung istirahat
  • walkingHeartRateAverage - Rata-rata detak saat berjalan
  • heartRateVariabilitySDNN - Variabilitas detak jantung (SDNN)
  • bloodPressureSystolic - Tekanan darah sistolik (mmHg)
  • bloodPressureDiastolic - Tekanan darah diastolik (mmHg)
  • oxygenSaturation - Saturasi oksigen (SpO₂ %)

Data ECG & Aritmia

  • HKElectrocardiogram - Rekaman ECG Apple Watch
  • irregularHeartRhythmEvent - Notifikasi ritme tidak teratur/AF

Metrik Kebugaran

  • vo2Max - Kapasitas aerobik (ml/kg/menit)
  • walkingSpeed - Kecepatan berjalan (m/detik)
  • walkingAsymmetryPercentage - Asimetri gaya berjalan (%)
  • stairAscentSpeed - Kecepatan naik tangga (langkah/detik)

Metrik Tubuh

  • bodyMass - Berat badan (kg)
  • height - Tinggi badan (m)
  • bodyMassIndex - BMI

Obat (Opsional)

  • HKMedicationRecord - Data obat dari Apple Health

Sinkronisasi Latar Belakang Otomatis

Cardio Analytics menggunakan fitur lanjutan HealthKit untuk menjaga data Anda tetap sinkron:

  • Kueri berlabuh - Hanya mengambil data baru sejak sinkronisasi terakhir (hemat baterai)
  • Pengiriman latar belakang - Menerima notifikasi ketika data baru tersedia
  • Pembaruan otomatis - Dashboard diperbarui bahkan saat aplikasi tidak berjalan

Kemampuan Penulisan Kembali

Cardio Analytics juga dapat menulis data kembali ke HealthKit:

  • Tekanan darah manual - Masukkan pembacaan dari monitor apa pun
  • Berat badan manual - Catat berat badan dari timbangan apa pun
  • Obat - Sinkronkan dosis obat ke Apple Health
  • Gejala - Dokumentasikan gejala dalam format HealthKit standar

Privasi & Keamanan

  • Izin terperinci - Anda mengontrol tepat jenis data mana yang dibagikan
  • Enkripsi - Data HealthKit dilindungi oleh enkripsi perangkat Apple
  • Penyimpanan lokal - Semua data tetap di perangkat Anda
  • Tidak ada transmisi cloud - Cardio Analytics tidak pernah mengirim data HealthKit ke server eksternal
