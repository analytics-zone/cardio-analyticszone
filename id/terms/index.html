Syarat dan Ketentuan

Terakhir diperbarui: Januari 2024

Pendahuluan

Syarat dan Ketentuan ini mengatur penggunaan Anda atas situs web kami. Dengan mengakses atau menggunakan situs web kami, Anda setuju untuk terikat dengan syarat-syarat ini.

Jika Anda tidak setuju dengan bagian mana pun dari syarat-syarat ini, Anda sebaiknya tidak menggunakan situs web kami.

Penggunaan Situs Web

Anda setuju untuk:

  • Menggunakan situs web hanya untuk tujuan yang sah
  • Tidak berusaha mendapatkan akses tidak sah ke bagian mana pun dari situs web
  • Tidak mengganggu fungsi situs web yang semestinya
  • Tidak mengirimkan kode yang berbahaya atau jahat
  • Menghormati hak kekayaan intelektual orang lain

Kekayaan Intelektual

Semua konten di situs web ini, termasuk teks, grafik, logo, gambar, dan perangkat lunak, adalah milik pemilik situs web atau pemberi lisensinya dan dilindungi oleh hak cipta dan undang-undang kekayaan intelektual lainnya.

Anda tidak diperbolehkan mereproduksi, mendistribusikan, memodifikasi, atau membuat karya turunan dari konten apa pun di situs web ini tanpa izin tertulis sebelumnya.

Penafian Jaminan

Situs web ini disediakan "apa adanya" tanpa jaminan apa pun, baik tersurat maupun tersirat. Kami tidak menjamin bahwa situs web akan tersedia setiap saat atau bahwa situs web akan bebas dari kesalahan atau virus.

Kami tidak memberikan jaminan tentang keakuratan, kelengkapan, atau keandalan informasi apa pun di situs web ini.

Pembatasan Tanggung Jawab

Sejauh yang diizinkan oleh hukum, kami tidak bertanggung jawab atas kerusakan tidak langsung, insidental, khusus, konsekuensial, atau hukuman yang timbul dari atau terkait dengan penggunaan Anda atas situs web ini.

Ini termasuk, tetapi tidak terbatas pada, kerusakan akibat kehilangan keuntungan, data, atau kerugian tidak berwujud lainnya.

Tautan Eksternal

Situs web kami mungkin berisi tautan ke situs web eksternal yang tidak dioperasikan oleh kami. Kami tidak memiliki kontrol atas konten dan praktik situs-situs ini dan tidak dapat menerima tanggung jawab atas kebijakan privasi atau konten mereka.

Perubahan Syarat

Kami berhak untuk mengubah Syarat dan Ketentuan ini kapan saja. Perubahan akan berlaku segera setelah diposting di halaman ini.

Penggunaan berkelanjutan Anda atas situs web setelah perubahan apa pun merupakan penerimaan Anda terhadap syarat-syarat baru.

Hukum yang Berlaku

Syarat dan Ketentuan ini diatur oleh dan ditafsirkan sesuai dengan hukum Spanyol, dan Anda secara tidak dapat ditarik kembali tunduk pada yurisdiksi eksklusif pengadilan di lokasi tersebut.

Pertanyaan?

Jika Anda memiliki pertanyaan tentang Syarat dan Ketentuan ini, silakan hubungi kami.
